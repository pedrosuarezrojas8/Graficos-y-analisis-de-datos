# Visualización de Datos

A comprehensive collection of Jupyter notebooks for learning and practicing data visualization techniques using Python libraries like Matplotlib and Seaborn.

## Repository Structure

### 📊 Matplotlib/
Learning materials covering Matplotlib fundamentals and advanced techniques:
- **Estilos visuales** - Visual styling options in Matplotlib
- **Tipos de gráficos** - Line, bar, scatter plots, histograms, boxplots, violinplots, and pie charts
- **Personalización** - Colors, lines, legends, annotations, subplots, and shared axes
- **Escalas y grillas** - Axis scales and grid configuration
- **Gráficos avanzados** - Heatmaps, contour plots, and 3D graphics

### 📈 Seaborn/
Advanced data visualization techniques using Seaborn:
- **Introducción** - Seaborn fundamentals and architecture
- **Distribuciones** - Displots, KDE plots, and time series visualization
- **Relaciones** - Jointplots and pairplots
- **Comparaciones** - Swarmplots for categorical data comparison
- **Regresión** - Regplots, LMplots, and residual plots
- **Matrices** - Heatmaps and clustermaps
- **Personalización** - Aesthetic customization and advanced parameters

### 📁 Proyectos de estudio/
Practice exercises and study projects:
- **Datasets**:
  - `diamonds.csv` - Diamond characteristics and prices
  - `housing.csv` - Housing market data
  - `melb_data.csv` - Melbourne real estate data
  - `train_clean.csv` - Cleaned training dataset

- **Exercises**:
  - Ejercicio-Practica-Diamantes.ipynb
  - Ejercicio-Practica-Housing.ipynb
  - Ejercicio-Practica-Titanic.ipynb
  - melbourne_practica.ipynb
  - Practica-EDA.ipynb
  - Titanic dataset visualization exercises
  - Boston Housing dataset exercises

## Getting Started

1. Ensure you have the required libraries installed:
   ```bash
   pip install matplotlib seaborn pandas numpy jupyter
   ```

2. Open any notebook with Jupyter:
   ```bash
   jupyter notebook
   ```

3. Start with the Matplotlib folder for fundamentals, then move to Seaborn for advanced techniques.

## Prerequisites

- Python 3.7+
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Learning Path

1. **Matplotlib Basics** → Start with `OD_VD1_ESP_M03_S09_T03` (Visual Styles)
2. **Chart Types** → Progress through S10 notebooks (Lines, Bars, Scatter, etc.)
3. **Customization** → Learn S11 materials (Colors, Legends, Subplots)
4. **Advanced Graphics** → Explore S12 (Heatmaps, 3D plots)
5. **Seaborn** → Move to Seaborn folder starting with S14 (Introduction)
6. **Practice** → Apply knowledge in `Proyectos de estudio/` exercises
